Server list

Verze: 2.2 Claim serveru a přepracovaný Votifier

mcservery.eu

4. dub 2023 4 min

Ahoj, dnes byla konečně vydána velká aktualizace server listu, která přináší mnoho významných změn po dlouhé době. Jak jsem slíbil při minulém menším updatu na začátku tohoto roku, adresy serverů byly změněny podle přání uživatelů. Ale to není všechno! Tuto novou funkci jsem si držel v tajnosti, ale nyní ji mohu konečně odhalit - je možné převést vlastnictví serveru bez kontaktování podpory. To by mělo usnadnit správu serverů pro naše uživatele. Pojďme se tedy podívat na všechny další úpravy.

⚡️ Načítání seznamu serverů bylo zrychleno o 70%

Je to zhruba přes rok, kdy byla provedena anketa kde jsem se uživatelů ptal, co by na mcservery.eu zlepšili. Častou odpovědí byla rychlost. Vím je to dlouhá doba, tato změna měli přijít dříve. Nicméně seznam serverů se již načítá rychleji, a to v průměru o 70 %

Doufám, že v dohledné době by se mohlo podařit toto načítání ještě zrychlit.

🔗 Mění se adresa stránky serveru

Další dlouho žádaná novinka, na detail serveru se nově lze dostat pomocí nového tvaru url adresy. např: mcservery.eu/server/jmeno-serveru

dochází tak k sjednocení stylu odkazů, s již dříve změněnými url uživatelských profilů.

Přístup přes parametr id prozatím zůstává stále funkční. Bude ale upřednostněno použití nových adres se jménem serveru.

📝 Oprava zobrazení popisů serverů

Již dříve bylo možné použití markdownu v popisu serveru, bohužel se ale některé styly nezobrazovali vždy správně. Tento problém by již měl být opraven.

Změnil jsem kód pro parsování Markdownu, nově tak na mcservery.eu používáme knihovnu Showdown.js a popisy by se již měli u všech serveru zobrazovat správně, tedy pokud v popisu správně používáte Markdown.

Je možné že bude u vašeho serveru nutné popis upravit, aby se zobrazoval tak jak má.

✅ Ověřený server

Nově se u serverů kde je přidán odkaz na web, a zároveň máte na Vašem webu zpětný odkaz na mcservery.eu bude zobrazovat že je váš server ověřený.

Potvrzujeme tak že uživatel který přidal daný server na server list, má pod kontrolou doménu kterou uvedl u serveru.

Propojené domény s vaším účtem si můžete zkontrolovat také v nastavení uživatelského účtu.

Více k ověřeným serverům: https://help.mcservery.eu/spravci-serveru/zpe-tny-odkaz/overeny-server

🗳️ Upravené hlasování pomocí Votifieru

Hlasování u serverů s nastaveným odměňováním za hlasování je nově uživatelsky přívětivější. Nick na serveru se upravuje v modalu, zároveň je defaultně nastaveno uživatelské jméno z webu, takže není nutné vyplňovat pokud se shoduje s přezdívkou na serveru pro který hlasujete.

Zároveň byla odstraněna chyba při editaci serveru, která mohla způsobit nefunkční propojení s Votifierem.

🔀 Převod serveru / vynucené vlastnictví

Pokud někdo přidal server, který je Váš, nebo jen chcete převést svůj server na jiného uživatele nově je to možné bez kontaktování podpory. Jedná se o ověření převodu vlastnictví serveru prostřednictvím kódu v motd serveru. Tuto funkci má mcservery.eu jako jediný server list na CZ/SK scéně.

K převodu serveru není nutná spolupráce uživatele, který server na server list přidal. Je však nutné mít přístup na FTP nebo mít jinou možnost, jak v administraci Vašeho hostingu změnit konfigurační soubor serveru.

Více k převodu serveru: https://help.mcservery.eu/spravci-serveru/prevod-vlastnictvi-serveru

Další změny:

  • Přes tagy na detailní stránce serveru se nově dá prokliknout na ostatní servery se stejnými tagy.
  • Drobné úpravy vzhledu stránek.
  • Odkaz z editace serveru zpět na stránku serveru.
  • Možnost kopírovat IP adresu serveru do schránky po kliknutí (zatím jen na detailu serveru).
  • Opět přepracovaný blog. Po 4x BTW 🤯
  • Oprava sitemap. Již se znovu automaticky generuje.

A to je pro zatím vše, v případě že by vám cokoliv nefungovalo nebo máte nějaký nápad, jak zlepšit mcservery.eu neváhejte se přidat na náš Discord server. Těším se na vaše připomínky a nápady jak tento projekt dál posouvat.
